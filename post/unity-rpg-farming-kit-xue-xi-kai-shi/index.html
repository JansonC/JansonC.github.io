Unity RPG Farming Kit学习开始

2019-05-04

这是一款Unity的插件，可以快速创建一款类似于《星露谷物语》的游戏，插件名《Unity RPG Farming Kit》，可以在unity商店中购买。

安装条件

  1. Unity 20181.2
  2. TextMeshPro 1.2.3
  3. Cinemachine 2.1.13
  4. 如果您看到编译器错误。将Scripting Runtime Version设置为4.x Equvalent

框架工具
框架提供的工具可以在Window> RPG Farm Framework Tools下找到它们

Automatic Character/Scene loading（可以使用你的玩家角色出现在测试任何场景）
Gameplay Scene Folder Name（游戏场景文件夹名称，任何游戏场景的文件夹名称）
Auto Load Scenes（自动加载列表中的场景）
Set Player To Camera（设置相机跟随玩家）
Give Player Item（给玩家物品，只需选择任何物品即可，然后点击Give选择框，游戏角色就会收到物品）
